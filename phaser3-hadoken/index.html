Phaser 3 Hadoken Demo

Select Keymap: Qwerty / Dvorak

Demo move list:

  • Hadoken: Down, Down+Forward, Forward, Light Punch
  • Huricane Kick: Down, Down+Backward, Forward, Light Kick
  • Summon Suffering: Down+Forward, Up+Backward, Forward, Down, Down+Forward, Down+Backward, Light Punch, Guard

phaser3-hadoken is an absurdly configurable library for handling fighting game style input matching in phaser. I consider it currently a work in progress as Gamepad support is not yet in place and introductory documentation is needed.

The core code should be relatively stable and is (mostly) ready for tinkering. As always I'm available on the phaser discord for questions and comments.
