Phaser 3 Hadoken Demo

Select Input: Keyboard / Gamepad
Select Keymap: Qwerty / Dvorak
Invert Axis: None / Horizontal / Vertical / Both

Demo move list:

  • Hadoken: Down, Down+Forward, Forward, Punch
  • Huricane Kick: Down, Down+Backward, Forward, Kick
  • Summon Suffering: Down+Forward, Up+Backward, Forward, Down, Down+Forward, Down+Backward, Light Punch, Guard

phaser3-hadoken is an absurdly configurable library for handling input in Phaser. It supports Keyboard and Gamepads out of the box and includes a solution for matching fighting game style move sequences.

The core code should be relatively stable and is (mostly) ready for tinkering. As always I'm available on the phaser discord for questions and comments.
